Favicon Guide for Mosaic Media

M

This guide provides instructions for creating the perfect favicon for the Mosaic Media website, matching our bold black and energetic yellow color scheme.

Creation Steps

  1. Take a screenshot of the "M" preview above or create a similar design
  2. Crop to a perfect square
  3. Resize to multiple standard sizes:
    • 16x16 pixels (browser tabs)
    • 32x32 pixels (taskbar, desktop)
    • 48x48 pixels (modern browsers)
    • 192x192 pixels (mobile home screens)
  4. Save as ICO format for maximum compatibility
  5. Place in the root directory as "favicon.ico"

HTML Implementation

Add these tags to the <head> section of all HTML pages:

<link rel="icon" type="image/x-icon" href="/favicon.ico">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">

Alternative Text-Based Favicon

For a minimalist approach, use the text "M" in our brand colors:

  • Background: Pure black (#000000)
  • Text: Energetic yellow (#FFD700)
  • Font: Inter Black (900 weight)
  • Centered with generous padding

Pro Tip: Test your favicon across different browsers and devices to ensure visibility against various background colors. The high contrast between black and yellow ensures excellent visibility in all contexts.
